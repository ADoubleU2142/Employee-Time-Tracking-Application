# Employee Time Tracking Application

Web application created to simplify employee working time registration for a small company.

## Overview

The goal of this project was to replace manual time tracking with a simple digital system that allows employees to record working hours and managers to review monthly summaries.

## Features

- PIN-based login
- Employee dashboard
- Work hours calendar
- Monthly summaries
- Absence management
- User-friendly interface designed for non-technical users

## Technologies

- HTML
- CSS
- JavaScript
- Bootstrap
- Supabase
- Git

## Project structure

The application is divided into:
- Components — reusable UI elements
- Views — application screens and logic
- Services — database communication
- Utilities — helper functions
- Core — shared state management

## Code examples

Selected examples of implementation:
- `state.js` — application state management
- `calendar.js` — dynamic calendar rendering
- `dashboard.js` — dashboard logic
- `entryService.js` — database operations
